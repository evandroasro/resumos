# Comandos Git Importantes

Comandos

``` sh
git init #criar repositório
git add #envia o arquivo
git commit #salva o arquivo
git push #envia para o repositório
git pull #puxa do repositótio
git clone #clona repositório
git rm #deleta arquivo
git log #varifica últimas alterações no projeto
git mv #renomeia ou move o arquivo
git checkout #desfaz alterações puxando o arquivo que está no repositório
git checkout -b nomedobranch #muda para o branch nomeado
git reset #reseta mudanças feitas
git branch #visualiza os branchs disponíveis
git branch nome #cria um novo branch
git branch -d nome #deleta 1 branch
git merge nome #une dois branchs
git stash #código é descartado para "lixeira"
git stash list #verifica todos os stashs criados
git stash show nome #recupera a stash nomeada
git stash clear #joga fora todas as stashs
git stash drop nome #joga fora a stash nomeada
git tag #cria tags nos branchs
git show nome #verifica tag
git push origin nome #envia a tag para o repositório
git fetch #atualização sobre todos os branchs e tags
git fetch -a #puxa todos os branchs
git remote #adiciona ou remove um repositório
git show #mostra informações
git diff # mostra diferenças entre branchs
git shortlog #log resumido do projeto
```
